
# 🚗 Road Accident Dashboard 

## 📌 Project Overview

The **Road Accident Dashboard** is an interactive **Microsoft Excel Dashboard** designed to analyze road accident data, casualties, vehicle involvement, weather conditions, and road types. Built using Excel's data analysis and visualization features, the dashboard provides a comprehensive overview of accident statistics through dynamic charts, KPI cards, and interactive filters, enabling users to identify trends and make data-driven decisions for road safety.

## 🎯 Objectives

* Monitor overall road accident statistics.
* Track total accidents, casualties, and vehicles involved.
* Analyze monthly accident trends.
* Identify accident-prone cities.
* Compare accidents across different weather conditions.
* Evaluate accident distribution by road type.

## 📊 Dashboard Features

### 🔹 Interactive Filters

* Year
* Weather Condition
* Road Type
* City

* 🥧 Accident by Weather
* 📈 Monthly Accident Trend
* 📊 Casualties vs Vehicles Involved
* 📊 Accident by Road Type

* Microsoft Excel

* Dashboard Design & Data Visualization

## 📈 Key Insights

* The dashboard identifies cities with the highest number of road accidents.
* Monthly accident trends help detect seasonal variations in accident frequency.
* Weather analysis highlights the impact of different weather conditions on road accidents.
* Urban roads record a higher number of accidents compared to other road types.
* Comparing casualties with vehicles involved provides insights into accident severity.
* Interactive filters enable detailed analysis by year, city, weather condition, and road type.
